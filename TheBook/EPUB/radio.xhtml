AM/FM Radio

Television is not the only source of news and entertainment that uses radio transmissions that are free to receive. AM and FM radio predate television considerably and are both still very much alive.

Please note that if you are only interested in FM radio, things are somewhat simpler. The FM band fallse between VHF-Lo and VHF-Hi and most modern radio tuners with antenna input use the same 75Ω coaxial female F-connector antenna jack that ATSC television uses.

This allows the same splitters and signal amplifiers that work with OTA television to also work with FM broadcast.

In fact for FM radio, if your antenna—such as many but not all Yagi-Uda antennas—works with both VHF-Lo and VHF-Hi there is a good possibility it also works well for FM, at least for FM broadcast from the same direction for which the antenna is pointed.

In theory you can use a combiner to combine the output of an FM antenna with your television antenna but in practice that is not very realistic. ATSC tuners do not decode FM signals (analog or digital) so you would lose about 4 dB when combining the signal only to lose about 4 dB again when you split the signal to feed both an ATSC tuner and an FM tuner. That is a 7–8 dB loss.

If you also want AM radio, AM broadcasts from 550–1720 kHz (0.55–1.72 MHz) which unfortunately is well below the operational range for amplifiers and splitters that are used for OTA television.

This appendix chapter will discuss setting up an AM/FM antenna distributed to multiple receivers via coaxial cable along with notes about setting up a similar system that is FM only.

Test Environment

An inexpensive ‘Antop AT-212FM Curved Panel Amplified Indoor AM/FM Antenna’ connected directly to the test receiver will be used as a reference antenna.

My analog only Yamaha R-S201 receiver will be used to test how many stations on both bands are received, and how well they come in. As such digital AM/FM reception will not be tested.

During the tests, my PC, television, and WiFi router will be turned off as they are all possible sources of interference except in the final test.

Due to physical logistics, the tests can not be performed all on the same day so it is possible atmospheric difference can impact the results. All tests will be performed between the hours of 1:00 A.M. and 3:00 A.M. when reception is typically very good.
