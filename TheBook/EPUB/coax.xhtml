Coaxial Cables and Accessories

For 75Ω impedence antenna systems, there are basically five different types of coaxial cable to consider. There are also several tools and accessories that you will need when working with coaxial cable runs.

Note that I am not a proffesional antenna installer but for this project I felt it important to both educate myself and create a basic toolkit for coaxial related work. For specific tasks there are other tools but the tools described here (after the coaxial cable types) are a good core set to always keep together.

RG-59

Historically, RG-59 was often used by professional installers because it was much cheaper than RG-6.

At 400 MHz, RG-59 loses about 7.0 dB of signal per 100 feet while RG-6 only loses about 4.3 dB. Now that RG-6 is much more affordable and widely available, RG-59 should generally be avoided for television and radio signals.

At very short distances where flexibility is needed—such as between your wall jack and your television—the signal loss difference will be very small which may make choosing RG-59 for that small distance an accessible trade-off for the increased flexibility.

RG-59 should not be used between the antenna and your wall jacks and really should only be used between a wall jack and a tuner when you just happen to have excess lying around. Even then it probably is better not to use it.

RG-59 Klein F Compression Connectors

If using RG-59, the correct Klein Tools F Compression Connectors will have a red band and will be labelled for RG59. Note that not all brands of F Compression Connectors follow the same color scheme that Klein Tools uses.

Dual-Shield RG-6

While not as flexible as RG-59, Dual-Shield RG-6 is much more flexible than Quad-Shield RG-6.

It often works just fine inside a wall but with longer runs Dual-Shield RG-6 is more susceptible to interference than Quad-Shield RG-6 which can degrade the Signal to Noise Ratio (SNR) potentially causing some reception issues.

Due to the increased flexibility, I recommend Dual-Shield RG-6 for the initial antenna location testing and for the short runs from the wall plate to the ATSC or radio receiver.

White is often a more aesthetic color coating for the run from the wall plate to the receiver and 100 feet is a good length for testing the antenna location.

You can by 100 feet of white Dual-Shield RG-6 for your antenna location test, and then repurpose it for the shorter runs from the wall plates to the tuners.

Dual-Shield Klein F Compression Connectors

If using Dual-Shield RG-6, the correct Klein Tools F Compression Connectors will have a blue band and will be labelled for RG6 but it NOT be labelled for RG6/6Q. Note that not all brands of F Compression Connectors follow the same color scheme that Klein Tools uses.

Quad-Shield RG-6 Copper-Clad

Quad-Shield RG-6 is far less susceptible to RF interference than Dual-Shield RG-6, though it is a little less flexible.

When only carrying an RF signal, there is no performance difference between a copper-clad steel core and a solid copper core, so Quad-Shield RG-6 Copper-Clad is what I recommend for most if not all of the coaxial cable runs between your antenna and the wall plates.

Even if you need more than 250 feet for the sum of all your runs, I recommend buying 250 feet pull boxes. They are much easier to work with than reels and the weight of 250 feet is still light enough to make it easy to climb up into your attic with it. Make sure a quality brand is used, and make sure the coaxial cable has marking so it is easy to see how much you used for a particular run. In the event you are running it in plenum space or in an HVAC duct, make sure it is plenum rated.

The Quad-Shield RG-6 Copper-Clad that I used is the Mediabridge Black (250 feet) Pull-Out Box. It is more expensive than the same length of other brands at the chain hardware stores but I wanted to reduce the odds of cable failure over time.

Quad-Shield RG-6 Klein Tools F Compression Connectors

If using Quad-Shield RG-6 (core type is irrelevant), the correct Klein Tools F Compression Connectors will have a yellow band and will be labelled for RG6/6Q. Note that not all brands of F Compression Connectors follow the same color scheme that Klein Tools uses.

Quad-Shield RG-6 Copper-Core

A solid copper core versus copper-clad steel core makes absolutely no difference for RF transmission over the coaxial cable.

However for the direct current used to power amplification, solid copper core does have less resistance than copper-clad steel and therefore is preferable though it is not critical. Amplifiers will still do their job even with copper-clad steel. Allegedly some sattelite dish controllers are touchy and require copper core if the run is over a certain length, check with your sattelite vendor for the RG-6 requirements if using RG-6 to carry a voltage to a sattelite dish controller. In the event you are running coaxial cable in plenum space or in an HVAC duct, make sure it is plenum rated.

The disadvantage to using copper-core is that it is more expensive than copper-clad steel.

For coaxial runs that carry DC voltage, I do recommend using Quad-Shield RG-6 Copper-Core but only if you can find it in lengths that are not excessive to your needs. Phat Satellite Intl sells it in various lengths on Amazon, e.g. 100 feet Quad Shield Solid Copper 3GHZ RG-6 Coaxial Cable is what I used for my DC voltage needs.

Quad-Shield RG-6 Klein Tools F Compression Connectors

If using Quad-Shield RG-6 (core type is irrelevant), the correct Klein Tools F Compression Connectors will have a yellow band and will be labelled for RG6/6Q. Note that not all brands of F Compression Connectors follow the same color scheme that Klein Tools uses.

RG-11

For long runs, RG-11 does have less signal loss than RG-6 but it is expensive, very thick, and extremely difficult to work with for any runs that are not straight.

For runs under 100 feet, the reflective properties of RG-11 may actually make it worse than RG-6.

For runs from 100 feet to 200 feet, the benefit is not worth the difficulty of working with RG-11, it usually better to overcome signal loss through signal amplification unless there really is not way to power an amplifier at the start of the run.

For runs that are over 200 feet in length and relatively straight, then RG-11 could be considered.

Unless you happen to live in a mansion, or if the best place for your antenna is on an out-building, RG-11 is almost certainly the wrong choice.

Signal Reflection

When a signal travels through a wire and meets any form of impedance or resistance, part of the signal will be reflected back. It is like water rushing in a creek that suddenly meets a bottleneck. Some of the water bounces back towards the source. Reflection can degrade the quality of a signal. The reflective distance is how far back the signal will reflect before it is reduced to below the noise floor.

The lower the signal loss qualites of a coaxial cable, the large that reflective distance is.

We often like to think of an RF signal traveling from the antenna to our television, but in reality its an oscillation that flows both ways. For this reason, a length of coaxial should be at least twice the reflective distance for that kind of coaxial cable. When a shorter length is needed between two connection points, the coaxial cable should be looped. Looping with RG-11 however is not realistic.

With RG-6 the reflective distance is about a foot so coaxial cable lengths of RG-6 should be at least two feet in length. I do not know the reflective distance of RG-59 but the two foot rule for RG-6 is a good rule of thumb as RG-59 will have a lower reflective distance. RG-11 has a fify foot reflective distance so lengths of RG-11 should be at least 100 feet. A damaged coaxial cable (kinked or squashed) could have a reflection point inside it, try to avoid damaging the coaxial cable.

Reflection is also why all runs that do not attach to a tuner and all unused outputs on splitter or amplified distributor should have a 75Ω termination cap. When a coaxial output or cable is not connected to something, it is an open circuit and the botleneck that causes reflection is where the run starts, not where it ends. This can result in reflective signal degration for the other runs that are in use. A 75Ω termination cap closes that open circuit reducing the reflection.

F Connectors and Coaxial Tools

F Connectors

In the United States, cabling generally uses what are called ‘F Connectors’ for 75Ω coaxial television and radio antenna connections. The coaxial cable will typically have a ‘Male F Connector’ attached to both ends. There are three methods of attaching a ‘Male F Connector’ to coaxial cable:

  1. ‘Push-On’ Connector
  2. ‘Screw-On’ Connector
  3. Compression Connector

Do not use the ‘Push-On’ connectors. They really are only intended for temporary use despite the fact that they are marketed to consumers as a solution that does not require tools to install. They come off too easily.

Do not the use ‘Screw-On’ connectors. They are a little better than the ‘Push-On’ but not much better. What they do is tap a thread into the jacket of the coaxial cable as they are screwed into place. When I lived in Redding and before I knew better, I used ‘Screw-On’ connectors for my coaxial runs and they did work, but they all eventually failed.

Do use the Compression connectors. Yes, they require an additional tool to properly install but they are far less likely to come apart.

For the compression connectors, they are often marketed as ‘Universal’ but it seems these sometimes are problematic for some users with dual-shield RG-6 and slip off if not used correctly. I recommend only using the the ‘Universal’ connectors with quad-shield unless you have no other choice.

Klein Tools red band F connectors will work with most RG-59 but will not work with RG-6. Klein Tools blue band connectors will work with most Dual-Shield RG-6 but is not likely to work with Quad-Shield RG-6. Klein Tools yellow-band connectors will are ‘Universal’ RG-6 and work with most Quad-Shield RG-6 and frequently will also work with Dual-Shield RG-6 but I only recommend it for Quad-Shield RG-6.

I personally recommend the Klein Tools compression connectors. However since I started using compression connectors I have not tried any other brands. I chose their compression connectors because I chose their tools and was more confident their connectors would work properly with their tools. There likely are other brands that work just as well.

I keep both Klein Tools ‘Blue’ and ‘Yellow’ connectors in a small hinged plastic box that snaps shut in my coaxial tool bag. I do not work with RG-59 so I do not have those compression connectors.

Coaxial Tools

Particularly when going into an attic without flooring I recommend having a tool bag specific for your coaxial tools. I use an inexpensibe Husky Bag that has a convenient shoulder strap.

For cutting coaxial cable you will need a good pair of cable cutters that can cut it quickly and cleanly. I personally use the Klein Tools Compact Cable Cutter. The size makes it easy to bring with you into the attic yet it still has to torque needed to make a nice clean professional cut.

Once you have the nice clean cut, before you can install the compression connector you have to strip the coaxial cable twice. The first strip larger strip removes the coating and then a short strip removes the dielectic from around the core wire.

It is tempting to use standard wire strippers to do this and it can be done that way but it really is better to use a tool specifically designed for the job. I use theKlein Tools 2-Level Coaxial Stripper. That tool makes it easy to get it right. There is a little extra work that needs to be done after using the tool.

Once the cable is ready for the compression connector. The tool I use to actually make the compression is the Klein Tools 9 inch Compact F-Connector Compression Crimper.. That particular tool only does F-connectors, they do make a larger one that also can be used for BNC and RCA> connectors that may be a better purchase if you expect to work with those kind of connectors too.

There is an excellent video on YouTube by ToolZine shows the process of using Klein Tools to install a compression F-connector on coaxial cable: www.youtube.com/watch?v=qo8KLJcrYiU.

7/16 Wrench

The nuts that tighten a male F connector to a female receptacle are SAE 7/16 inch. You can sometimes get away with using a metric 12mm but it will not be a tight fit, you should have an actual 7/16 inch wrench and it should have a short handle both to avoid too much torque and to allow it to be used in tight places.

The Tekton 7/16 inch Stubby Combination Wrench is a good choice. Keep one with your coaxial tools.

While not often needed, an angle-head variant can sometimes be beneficial: Tekton 7/16 inch Angle Head Open End Wrench.

IMPORTANT: Never over-tighten the nut on an F connector or you can break something. Use your fingers to hand tighten it and then use the wrench to just tighten it barely a tad bit more.

Screwdrivers

A #1 x 3 inch phillips screwdriver is really handy for mounting splitters and amplified distributors. A #2 x 4 inch phillips screwdriver is frequently beneficial for antenna installations. You likely already have equivalents but they are inexpensive and it is good to have some to specifically always keep these with your coaxial specific tools.

Connectivity Testing Tool

This Klein Tools Coax Explorer 2 is not absolutely needed but when there is a problem, it can be very useful. I keep this in the tool bag but remove it when I will not be using it to reduce the clutter.

The tool has a remote that you place on one end of a coaxial run, and you then connect the tester to the other end. Push a button, and you can verify the coaxial cable works.

As with any tool with batteries, remove the batteries when you will not be using it for awhile.

They also sell a Coax Explorer Remote Kit that has four different color coded remotes (one duplicates the remote that comes with the tool). This remote kit is very handy for testing several runs at once. Simply put a remote on each wall plate that comes off of a splitter and use the test on the splitter. You can then see which coaxial runs have a connection and which runs have a problem.

Coaxial Accessories

With the exception of the test cable listed at the end, I like to keep some of these in my tool bag at all times. Like compression connectors, I keep these in small plastic hinged boxes that snap shut.

Right Angle Connectors

Coaxial cable does not like to make sharp bends. In many cases, installation of a cable can be greatly simplified if a right angle (90°) connector is used. Televisions and wall-plates are a common place where there are beneficial.

Amazon has them in five packs: VCE 5-Pack 90 Degree Coaxial Connector. I am not convinced the gold plating has real-world benefit beyond marketing.

Home Depot has them in two packs: Ideal 90° Coaxial F-Type Adapters.

When installing these, make sure the connection is tight but do be careful not to over-tighten.

Barrel Connectors

Barrel connectors have two female connections and are used to splice to different pieces of coaxial cables together. A coaxial wall insert for example is just a barrel connector mounted in plastic and is used to splice the run inside the wall with the run to your tuner (or to a splitter etc.)

It is a good idea to always have some spare barrel connectors on hand.

Amazon has them in five packs: VCE 5-Pack Coaxial Cable Connector. I am not convinced the gold plating has real-world benefit beyond marketing.

Home Depot has them in ten packs: Klein Tools 3 GHz F-Splice Adapter.

Frequently one end is a little longer than the other. The only reason for that is allow it to easily be used in a wall plate, the longer one is placed through a hole with a really thin nut threaded over the longer end on the other side of of the plate to hold the connector to the plate.

75Ω Terminator Caps

Any wall plate that is not currently connected to a tuner should be terminated with a 75Ω Terminator or the signal to active devices could be degraded. This also applies to unused outputs on splitters and amplified distributors.

Amazon has them in 20 packs: Ancable 20-Pack F Type Terminator Caps with 75 Ohm Resistor.

When you need to terminate the end of a coaxial cable with a male F connector, you will need to use a barrel connector between the coaxial cable and the barrel connector.

When I install these, I only hand-tighten them as they are not likely to loosen by themselves.

Mounting Screws

For splitters and distribution amplifiers in the attic I prefer to mount them to the side of a joist or another beam using #6 x ¾ inch wood screws. It is always good to have spare screws at all times in case I have to unmount a splitter and I drop the screw into the loose insulation.

When making a fresh install with #6 wood screws, drill a 9/64 (⅛ for soft wood) inch pilot hole first to avoid splitting the wood. See Wood Screw Pilot Hole Sizes.

Spare 10 foot Quality Quad-Shield RG-6 Cable

Obviously I do not keep this in my tool bag.

If you are like me, the flexibility of dual-shield RG-6 makes it a very attractive cable to use from the wall jack to the tuner. It usually works but there are cases where that results in the signal degrading enough that some channels have tuning issues if their signal is borderline to begin with. Even if you do use quad-shield RG-6 for those runs, it is still possible for such a cable to fail, especially if it ever has been squished or kinked.

It is a good idea to have a pre-made 10 foot quad-shield RG-6 test cable that you can use when diagnosing a tuner that has problems tuning a signal to see if the run from the wall jack to the tuner is to blame.
